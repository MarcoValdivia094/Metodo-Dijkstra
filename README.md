# Simulador del Algoritmo de Dijkstra

Este proyecto incluye:
- **dijkstra_console.py**: simulación paso a paso en consola.
- **dijkstra_graph.py**: visualización gráfica con NetworkX y Matplotlib.

## Cómo ejecutar en Google Colab
1. Copia el código en celdas de Colab.
2. Instala librerías con `!pip install networkx matplotlib`.
3. Ejecuta las celdas en orden.
